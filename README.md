# Estedad-MDarvishi

This fork is an Urdu-compatible modification of [Estedad-MDarvishi](https://github.com/MohamadDarvishi/Estedad-MDarvishi) by [MohammadDarvishi](https://github.com/MohamadDarvishi).

I modified this font for my personal use on [my website](https://src4026.gitlab.io) for some of the textual matter in Urdu.

Due to personal stylistic and convenience as well as my lack of experience, knowledge, resources, and good practice. I have only kept the dot-style 4 from the fork and the semibold mid weight, along with some additional darker variants as I found suitable.
