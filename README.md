# Estedad-MDarvishi

This fork is an Urdu-compatible modification of the [Estedad](https://github.com/aminabedi68/Estedad) font's [fork](https://github.com/MohamadDarvishi/Estedad-MDarvishi) by [MohammadDarvishi](https://github.com/MohamadDarvishi).

I modified this font for my personal use on [my website](https://src4026.gitlab.io) for my post headings that are in Urdu.

Due to personal stylistic and convenience, I have only kept the dot-style 4 and mid variants of all the static fonts.
